Smart Voting System using Verilog
Overview

This project is a digital voting machine designed using Verilog HDL. The system allows users to cast votes securely, counts votes for candidates, and prevents duplicate voting by ensuring each voter can vote only once.

Features
>Digital vote counting
>Duplicate vote prevention
>Secure voting process
>Candidate-wise vote counting
>Hardware simulation using Verilog

Technologies Used
>Verilog HDL
>FSM (Finite State Machine)
>ModelSim
>Xilinx Vivado
>Digital Logic Design

Working
1.User enters vote
2.System checks voter validity
3.If voter already voted → vote rejected
4.If valid → vote counted
5.Final vote count displayed

Applications
Electronic Voting Machines
Secure Election Systems
Digital Logic Design Projects
